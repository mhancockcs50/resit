I first set up the project with a C99 CMake build configuration and a .gitignore file to create the basic development structure and prepare the project for development.
Before writing any code, I had to install and configure GCC through MSYS2 and CMake because my laptop's Killer network drivers became corrupted. I had to factory reset the laptop about two weeks beforehand, as the newer Intel Killer software was incompatible with my machine.
I then created the header files (adc.h, io.h, and stats.h) to define the project's data structures, constants, and function prototypes. This included the packed ADC record and file header structures used to match the binary log format, along with the structures required to store processed sample data and analysis results.
Next, I implemented the generic statistics functions in stats.c, including routines to calculate the mean, minimum, maximum, population standard deviation, and RMS value. These functions were written independently of the ADC-specific code so they could be reused with any array of numerical data.
